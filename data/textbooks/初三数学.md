## 第21章 一元二次方程

### 21.1 一元二次方程的概念

只含有一个未知数（一元），且未知数的最高次数是2（二次）的整式方程，叫做**一元二次方程**。

一般形式：ax² + bx + c = 0（a ≠ 0）

其中 ax² 是二次项，a 是二次项系数；bx 是一次项，b 是一次项系数；c 是常数项。

**知识点1：** 判断一元二次方程的三个条件：①整式方程 ②一个未知数 ③最高次数为2

### 21.2 解法（一）——直接开平方法

对于 x² = p（p ≥ 0），解得 x = ±√p。适用于缺少一次项或可化为完全平方的方程。

**例：** 解 x² - 25 = 0，得 x = ±5

### 21.3 解法（二）——配方法

步骤：①二次项系数化为1 ②移常数项 ③两边加上一次项系数一半的平方 ④写成完全平方 ⑤开平方

**口诀：** 一化二移三配四写五开

**例：** x² - 6x + 4 = 0，配方得 (x-3)² = 5，x = 3 ± √5

### 21.4 解法（三）——公式法

求根公式：x = (-b ± √(b²-4ac)) / 2a（a ≠ 0）

**判别式** Δ = b² - 4ac：
- Δ > 0：两个不等实根
- Δ = 0：两个相等实根
- Δ < 0：无实根

**知识点2：** 求根公式是通法，任何一元二次方程都可用公式法求解

### 21.5 解法（四）——因式分解法

将方程化为 (x-x₁)(x-x₂)=0 的形式，得 x=x₁ 或 x=x₂。

常用方法：提公因式法、公式法（平方差/完全平方）、十字相乘法

### 21.6 韦达定理（根与系数关系）

若 x₁, x₂ 是 ax²+bx+c=0 的两根，则：
- x₁ + x₂ = -b/a
- x₁ · x₂ = c/a

**知识点3：** 韦达定理常用于不求根而进行根的运算，如求 x₁²+x₂²、1/x₁+1/x₂ 等

### 21.7 一元二次方程的应用

列方程解应用题步骤：审→设→列→解→验→答。常见题型：增长率问题、面积问题、利润问题、数字问题。

## 第22章 二次函数

### 22.1 二次函数的概念

形如 y = ax² + bx + c（a ≠ 0）的函数叫做**二次函数**。

**知识点1：** a 决定开口方向和大小，a>0 开口向上，a<0 开口向下，|a| 越大开口越窄

### 22.2 二次函数的图像与性质

图像是**抛物线**，对称轴 x = -b/(2a)，顶点 (-b/(2a), (4ac-b²)/(4a))

| 形式 | 表达式 | 顶点 |
|------|--------|------|
| 一般式 | y = ax²+bx+c | (-b/(2a), (4ac-b²)/(4a)) |
| 顶点式 | y = a(x-h)²+k | (h, k) |
| 交点式 | y = a(x-x₁)(x-x₂) | ((x₁+x₂)/2, f((x₁+x₂)/2)) |

**知识点2：** 三种形式可互相转化，配方是化一般式为顶点式的关键

### 22.3 二次函数与一元二次方程

二次函数 y=ax²+bx+c 与 x 轴交点个数 = 方程 ax²+bx+c=0 的实根个数：
- Δ>0：两个交点
- Δ=0：一个交点（顶点在 x 轴上）
- Δ<0：无交点

### 22.4 二次函数的应用

最值问题、抛物线型实际问题（拱桥、喷泉、投篮轨迹等）

**知识点3：** 实际应用中自变量的取值范围往往有限制，需结合实际意义

## 第23章 旋转

### 23.1 旋转的概念

将一个图形绕某点转动一定角度的变换。旋转中心、旋转方向、旋转角度是旋转三要素。

**知识点1：** 旋转不改变图形的形状和大小（保距变换）

### 23.2 旋转的性质

- 对应点到旋转中心距离相等
- 对应点与旋转中心连线夹角等于旋转角
- 旋转前后的图形全等

### 23.3 中心对称

绕某点旋转180°后与自身重合的图形。中心对称图形的对称中心是对应点连线的中点。

**知识点2：** 平行四边形、矩形、菱形、正方形、圆都是中心对称图形

### 23.4 关于原点对称的点的坐标

点 P(x, y) 关于原点对称的点为 P'(-x, -y)

## 第24章 圆

### 24.1 圆的有关概念

**圆：** 到定点距离等于定长的点的集合。圆心 O，半径 r。

弦、直径（最长的弦）、弧（优弧/劣弧）、等圆、等弧

**知识点1：** 直径是弦，但弦不一定是直径；直径是圆中最长的弦

### 24.2 垂径定理

垂直于弦的直径平分弦，且平分弦所对的两条弧。

推论：平分弦（非直径）的直径垂直于弦，且平分弦所对的两条弧。

### 24.3 圆心角与圆周角

- **圆心角定理：** 同圆或等圆中，相等的圆心角所对的弧相等、弦相等
- **圆周角定理：** 同弧所对圆周角 = 圆心角的一半
- **推论：** 直径所对圆周角是90°（直角）

**知识点2：** 同弧所对的圆周角相等；等弧所对的圆周角相等

### 24.4 点与圆的位置关系

设圆心 O，半径 r，点 P 到 O 距离为 d：
- d < r：点在圆内
- d = r：点在圆上
- d > r：点在圆外

**三角形的外接圆：** 三边垂直平分线交点（外心）到三顶点距离相等

### 24.5 直线与圆的位置关系

设圆心到直线距离为 d：
- d < r：相交（两个交点）→ 弦
- d = r：相切（一个交点）→ 切线
- d > r：相离（无交点）

**切线的判定与性质：** 过半径外端且垂直于该半径的直线是切线

**切线长定理：** 从圆外一点引两条切线，切线长相等

**三角形的内切圆：** 三角平分线交点（内心）到三边距离相等

### 24.6 弧长与扇形面积

- 弧长：l = nπr / 180
- 扇形面积：S = nπr² / 360 = (1/2) lr
- 圆锥侧面积：S侧 = πr l（l 为母线长）

## 第25章 概率初步

### 25.1 随机事件

- **必然事件：** 一定发生（概率 = 1）
- **不可能事件：** 一定不发生（概率 = 0）
- **随机事件：** 可能发生也可能不发生（0 < 概率 < 1）

### 25.2 概率的定义

P(A) = m / n，其中 n 为所有等可能结果总数，m 为事件 A 包含的结果数。

**知识点：** 用列举法（列表法/树状图法）求概率，适用于有限等可能事件

### 25.3 用频率估计概率

大量重复试验中，事件发生的频率稳定于某个常数，即概率。

## 第26章 反比例函数

### 26.1 反比例函数的概念

形如 y = k/x（k ≠ 0）的函数。x 的取值范围是 x ≠ 0。

**知识点1：** k>0 时图像在一三象限，k<0 时在二四象限

### 26.2 反比例函数的图像与性质

图像是**双曲线**，关于原点中心对称，关于 y=±x 轴对称。

- k > 0：在每个象限内 y 随 x 增大而减小
- k < 0：在每个象限内 y 随 x 增大而增大

**知识点2：** |k| 等于双曲线上任意一点向坐标轴作垂线所围矩形面积

### 26.3 反比例函数的应用

压强与体积（P=F/S）、欧姆定律（I=U/R）、速度与时间（v=s/t）

## 第27章 相似

### 27.1 相似图形的概念

形状相同、大小不一定相同的图形。相似比（相似系数）k 是对应边的比值。

### 27.2 相似三角形的判定

| 判定方法 | 条件 |
|---------|------|
| SSS | 三边对应成比例 |
| SAS | 两边成比例且夹角相等 |
| AA | 两角分别相等 |
| HL | 斜边和直角边成比例（直角三角形）|

**知识点1：** 相似三角形的对应高、中线、角平分线的比都等于相似比

### 27.3 相似三角形的性质

- 对应角相等，对应边成比例
- 周长比 = 相似比
- 面积比 = 相似比的平方

### 27.4 位似

两个相似图形对应点连线交于一点（位似中心）。位似是特殊的相似变换。

**知识点2：** 位似图形上任意一对对应点到位似中心的距离之比等于位似比

## 第28章 锐角三角函数

### 28.1 锐角三角函数的定义

在直角三角形 ABC 中，∠C = 90°：

- sinA = 对边/斜边 = a/c
- cosA = 邻边/斜边 = b/c
- tanA = 对边/邻边 = a/b

**知识点1：** sinA = cos(90°-A)，tanA = sinA/cosA

### 28.2 特殊角的三角函数值

| 角度 | sin | cos | tan |
|------|-----|-----|-----|
| 30° | 1/2 | √3/2 | √3/3 |
| 45° | √2/2 | √2/2 | 1 |
| 60° | √3/2 | 1/2 | √3 |

**口诀：** 三十四五六十度，三角函数记牢固

### 28.3 解直角三角形

已知一边一角或两边，求其余边和角。

**知识点2：** 解直角三角形只需知道两个条件（至少一边），运用勾股定理和三角函数

### 28.4 应用举例

测高问题（仰角/俯角）、测距问题、航海问题（方向角）、坡度问题（i = h/l = tanα）

## 第29章 投影与视图

### 29.1 投影

- **平行投影：** 太阳光线（平行光线）
- **中心投影：** 灯光（从一点出发）

**知识点：** 同一时刻物高与影长成正比（平行投影）

### 29.2 三视图

- **主视图：** 从正面看
- **俯视图：** 从上面看
- **左视图：** 从左面看

**知识点：** 三视图遵循"长对正、高平齐、宽相等"原则

### 29.3 由三视图还原几何体

根据三视图想象并确定几何体的形状和尺寸，计算表面积和体积。
